# `libs/contracts`

Thu muc nay chua API contract va event contract chia se giua cac service.

## Trang thai hien tai

- Da co `openapi/gateway.yaml`
- Thu muc `events/` da duoc tao nhung chua co schema

## Noi dung dang co

- `openapi/gateway.yaml`
  - OpenAPI 3.0.3 toi thieu cho route `GET /health` cua gateway

## Vai tro cua folder nay

- Chot contract truoc khi implement service
- Giu tai lieu API o mot noi chung thay vi nam tung noi trong source
- Tao nen tang de sinh type/client ve sau
