# Thu Muc `libs`

Thu muc nay danh cho ma dung chung va contract chia se giua cac service/app.

## Trang thai hien tai

- `auth/`
  - Moi la placeholder cho helper auth va role constant dung chung
- `contracts/`
  - Da co OpenAPI toi thieu cho gateway
  - Thu muc event schema da duoc tao nhung chua co noi dung

## Nguyen tac su dung

- Dat code chia se that su, khong copy-paste giua cac service
- Contract API/event nen duoc dinh nghia tai day de giu ranh gioi giua cac service ro rang
