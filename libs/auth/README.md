# `libs/auth`

Thu muc nay du kien chua ma auth dung chung cho toan repo.

## Trang thai hien tai

- Chua co source code
- Chua co helper hay role constant duoc chia se

## Vai tro du kien

- Chua role constant dung chung
- Chua helper doc token/claim
- Chua guard/decorator co the tai su dung giua cac service

## Khuyen nghi

Khi bat dau tai cau truc auth dung chung, nen dua vao day:

- enum/constant cho role
- utility doc realm role tu Keycloak token
- helper tao auth header trong test
