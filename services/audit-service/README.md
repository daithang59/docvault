# `audit-service`

Day la folder placeholder cho audit service.

## Trang thai hien tai

- Chua co source code
- Chua co schema audit event
- MongoDB da san sang trong `infra/` nhung service nay chua ton tai

## Vai tro du kien

- Ghi audit event append-only
- Cung cap query cho role `co` hoac `admin`
- Tong hop lich su hanh dong nhu:
  - tao document
  - upload
  - submit
  - approve/reject
  - download allow/deny

## Ghi chu

Use case "CO xem audit duoc nhung bi cam download" trong tai lieu roadmap se phu thuoc rat nhieu vao service nay, nhung hien tai repo chua implement.
