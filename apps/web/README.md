# `apps/web`

Day la folder du kien cho frontend cua DocVault.

## Trang thai hien tai

- Hien chi co `.gitkeep`
- Chua co app Next.js/React
- Chua co login, dashboard, document list hay UI cho workflow

## Vai tro du kien

- Dang nhap qua Keycloak
- Goi API thong qua `gateway`
- Hien thi document list, detail, approval va audit theo role

## Cach su dung

Khi bat dau implement frontend, hay scaffold app ngay trong thu muc nay va cap nhat README voi:

- stack duoc chon
- lenh chay local
- bien moi truong can thiet
